Relatório de Estágio

1. Introdução

1.1. Objetivos

1.1.1. Objetivo Geral

Desenvolver habilidades práticas na área de desenvolvimento web por meio da participação em atividades de programação, manutenção de sistemas e suporte técnico em um ambiente profissional, aplicando os conhecimentos adquiridos ao longo da formação acadêmica.

1.1.2. Objetivos EspecIficos

  1. Participar do desenvolvimento e manutenção de aplicações web.
  2. Colaborar com equipes de desenvolvimento utilizando metodologias ágeis.
  3. Adquirir conhecimento prático sobre ferramentas de versionamento como Git.
  4. Aprimorar habilidades em linguagens como HTML, CSS, JavaScript e frameworks como React.js.
  5. Compreender o ciclo de vida de um projeto web, desde a concepção até a publicação.

2. Atividades Realizadas

2.1. Desenvolvimento de Interface de Usuário com React.js

Durante o estágio, participei ativamente da construção e melhoria de interfaces web utilizando o framework React.js. As atividades incluíram:

  1. Criação de componentes reutilizáveis;
  2. Consumo de APIs REST para exibição de dados dinâmicos;
  3. Estilização de componentes utilizando CSS Modules e Styled Components;
  4. Correção de bugs de layout e melhoria da responsividade das páginas.

Além disso, participei de reuniões diárias com a equipe (daily meetings), utilizando metodologias ágeis como Scrum para organização das tarefas.

2.2. Integração com APIs e Manutenção de Sistema Interno

Outra atividade relevante foi a integração de novas funcionalidades ao sistema interno da empresa:

  1. Implementação de funcionalidades como filtros de busca e ordenação de dados;
  2. Integração com APIs externas utilizando Axios;
  3. Testes unitários com Jest para garantir a estabilidade do sistema;
  4. Colaboração no controle de versão dos códigos via Git e GitHub.

Também acompanhei atividades de deploy utilizando plataformas como Vercel e Netlify.

3. Conclusão

O estágio foi uma oportunidade valiosa para aplicar na prática os conhecimentos adquiridos em sala de aula e aprofundar o entendimento sobre o desenvolvimento web moderno. Além de aprender ferramentas e linguagens amplamente utilizadas no mercado, também desenvolvi habilidades interpessoais importantes, como trabalho em equipe, comunicação e resolução de problemas.

A experiência reforçou meu interesse pela área de desenvolvimento front-end e ampliou minha visão sobre os desafios e as exigências do mercado de trabalho na área de tecnologia da informação.
